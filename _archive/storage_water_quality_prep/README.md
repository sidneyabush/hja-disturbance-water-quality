# Archived Storage-Water-Quality Prep

These scripts were part of the broader storage-water-quality workflow:

- hydrologic seasons
- rolling C-Q windows
- chemistry clusters
- synchrony calculations
- combined master tables
- storage-framework joins

They are archived here because the disturbance paper currently uses a leaner workflow:

1. Build `HJA_CQ_master.csv`.
2. Build annual disturbance chemistry summaries.
3. Join fire and flood disturbance drivers.
4. Make preliminary disturbance-paper figures.

Keeping these scripts out of active Step 01 avoids implying that clusters, hydroseasons, synchrony, or storage-framework ordinations are required for the disturbance paper.
